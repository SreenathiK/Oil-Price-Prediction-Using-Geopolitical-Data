Oil Price Prediction Using Geopolitical Data

Project Overview:
This project predicts Brent crude oil prices using historical geopolitical and economic data. A machine learning model was trained on past data to forecast oil prices and compare the predicted values with the actual market prices.

The model achieved an accuracy of approximately 96% on the evaluation dataset.

Features:
Data preprocessing and cleaning
Machine Learning model for oil price prediction
Prediction of Brent crude oil prices
Performance evaluation
Visualization of Actual vs Predicted prices using Matplotlib

Technologies Used:
Python
Jupyter Notebook
Pandas
NumPy
Scikit-learn
Matplotlib

Project Workflow:
Load historical geopolitical and oil price data.
Preprocess and clean the dataset.
Split the data into training and testing sets.
Train the machine learning model.
Predict Brent oil prices.
Evaluate model performance.
Visualize actual vs predicted prices.
Results
Model Accuracy: 96%
